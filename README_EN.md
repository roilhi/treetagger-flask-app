# 🧠 LCR Lab Project: Text Processing and Tagging
The LCR Lab project is a comprehensive text processing and tagging system that utilizes the Flask web framework, MongoDB database, and TreeTagger library to analyze and categorize text data. The project aims to provide a robust and efficient solution for text processing, enabling users to extract valuable insights from large datasets. The core features of the project include text tagging, part-of-speech (POS) tagging, and data storage in a MongoDB database.

## 🚀 Features
- **Text Tagging**: The project uses the TreeTagger library to tag and count the occurrences of each word and POS in a given text file.
- **Data Storage**: The project stores the tagged data in a MongoDB database, allowing for efficient retrieval and analysis.
- **Web Application**: The project includes a Flask web application that provides a user-friendly interface for uploading text files and viewing the tagged data.
- **Responsive Design**: The web application features a responsive design, ensuring a seamless user experience across various devices and screen sizes.

## 🛠️ Tech Stack
* **Frontend**: HTML, CSS, Bootstrap
* **Backend**: Flask, Python
* **Database**: MongoDB
* **Text Processing**: TreeTagger
* **Data Analysis**: Pandas, NumPy, SciPy, scikit-learn
* **Containerization**: Docker
* **Orchestration**: Docker Compose

## 📦 Installation
To install the project, follow these steps:
1. Clone the repository using `git clone`.
2. Install the dependencies using `pip install -r requirements.txt`.
3. Build the Docker image using `docker build -t lcr-lab`.
4. Run the Docker container using `docker run -p 5000:5000 lcr-lab`.

## 💻 Usage
To use the project, follow these steps:
1. Upload a text file to the web application.
2. The application will tag and count the occurrences of each word and POS in the text file.
3. The tagged data will be stored in the MongoDB database.
4. The user can view the tagged data by accessing the web application.
